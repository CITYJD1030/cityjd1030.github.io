AI 無線通訊系統實驗課

AI Lab for Wireless Communication

National Yang Ming Chiao Tung University
  • Dept. of course offering: Department of Electrical and Computer Engineering
  • Required/Elective: Elective
  • Instructor: 黃昱智、方凱田、伍紹勳

本課程透過實作方式介紹無線通訊技術與人工智慧演算法，涵蓋通道解碼、資源分配與雷達感測等主題，並強調 AI 在無線通訊中的實務應用。
This course introduces wireless communication technologies and AI algorithms through hands-on projects, focusing on practical applications such as channel decoding, resource allocation, and radar sensing.

在本課程中，我透過 Python 實作多種 AI 演算法在無線通訊場景中的應用，內容包含通道解碼、熱點定位、網路資源分配與雷達感測等。
In this course, I implemented various AI algorithms in Python to address real-world wireless communication problems such as channel decoding, hotspot localization, resource allocation, and radar sensing.

課程前段聚焦於通道解碼技術，從傳統演算法延伸至以 SVM 和深度學習為基礎的分類方法，並完成相關專題實作。
The early part of the course focused on channel decoding, moving from traditional decoding algorithms to classification-based methods using SVMs and deep learning, culminating in a dedicated project.

中段課程介紹 Q-Learning 與 DQN，並實作於多 UE 的網路資源分配問題中，藉此了解 AI 強化學習在動態系統中的應用潛力。
The middle section explored Q-Learning and DQN for multi-user downlink resource allocation, highlighting reinforcement learning's adaptability in dynamic environments.

後段課程延伸至雷達感測與生成模型，如 GAN、VAE、LSTM 與 CNN 在訊號解調、角度估計與室內定位的應用，並以多項專題作為整合驗證。
The later part extended into radar sensing and generative modeling, where GANs, VAEs, LSTMs, and CNNs were used for signal demodulation, angle of arrival estimation, and indoor localization, culminating in several final projects.

📂 課程資料與成果

    比較多種通道解碼方法（Syndrome, ML, DNN, Autoencoder）於 AWGN 通道下的 BLER 表現。
    Compare decoding methods (Syndrome, ML, DNN, Autoencoder) on AWGN channel based on BLER performance.
  • Module 2 - Lab 1: MNIST Fully Connected Classifier
    使用全連接神經網路訓練 MNIST 手寫數字辨識任務，達到近 98% 測試準確率。
    Trained a fully connected neural network for MNIST digit classification, achieving ~98% accuracy.
  • Module 2 - Lab 2: CNN-based Indoor Localization
    比較多種 CNN 架構於室內定位任務的效果，加入 BatchNorm、Dropout 與 Pooling 增強泛化能力。
    Compared CNN designs for indoor localization with added BatchNorm, Dropout, and Pooling to improve generalization.
  • Module 2 - Lab 3: GAN for Data Generation
    使用 GAN 產生擬真資料增強模型判別能力，並探討對抗訓練流程與應用。
    Used GAN to generate synthetic samples for enhancing classification, and explored adversarial training processes.
  • Module 2 - Lab 4: AE-GAN for Signal Augmentation
    將 Autoencoder 結合 GAN 用於訊號資料擴增，改善模型泛化與穩定性。
    Combined Autoencoder and GAN for signal augmentation to enhance model generalization and stability.
  • Module 2 - Lab 5: Q-Learning Resource Allocation
    實作 Q-Learning 演算法於多 AP 多 UE 環境中的頻道與功率資源最佳化配置。
    Implemented Q-Learning for optimal frequency and power allocation in multi-AP, multi-UE environments.
  • Module 3 - CNN-based Multi-direction Prediction
    使用 CNN 模型預測雷達目標方向，設計多卷積層與 Exact Match Accuracy 評估指標。
    Designed a CNN to predict radar target directions with Exact Match Accuracy for multi-label classification.

用Python於深度學習

Labs on Python for Deep Learning

National Yang Ming Chiao Tung University
  • Dept. of course offering: Department of Electrical and Computer Engineering
  • Required/Elective: Elective
  • Instructor: 李冕

使用Python完成多種深度學習模型應用，並透過各種實作、分析與比較，了解各種有關深度學習的相關知識。以下為課程的Challenge與我的成果：

Use Python to complete various deep learning model applications, and learn various deep learning related knowledge through various practices, analysis and comparisons. The following are the challenges of the course and my achievements:

  • Challenge 1: Activation Functions for Function Upscaling
    比較不同 activation function（ReLU, GELU 等）在函數近似與外推的效能差異。
    Compare the performance differences of different activation functions (ReLU, GELU, etc.) in function approximation and extrapolation.
  • Challenge 2: Gradient Descent Variants
    探索 Adam、Momentum、SGD 等學習演算法在不同 loss 曲面上的收斂速度。
    Explore the convergence speed of learning algorithms such as Adam, Momentum, SGD on different loss surfaces.
  • Challenge 3: Padding Effects in CNN/RNN/GRU
    比較不同 padding 策略對 NLP 模型的 F1 / AUC 表現。
    Compare the F1 / AUC performance of different padding strategies on NLP models.
  • Challenge 4: Word Embedding with BERT
    以 BERT 模型實作問答與分類任務，理解其 embedding 與 attention 機制。
    Implement question answering and classification tasks using the BERT model to understand its embedding and attention mechanisms.
  • Challenge 5: Dissecting Attention Mechanisms and Training Transformers

    This project explores attention in Transformers and builds a translation model.
  • Challenge 6: Embedding Edit Distance with Variational Autoencoders
    使用 VAE 建構潛在空間以反映二進位序列的編輯距離。
    Use VAE to embed binary sequences, making latent distances reflect edit distances.

在這門課中，我透過 Python 與 PyTorch 實作多種統計學與深度學習模型，內容涵蓋從基礎函數擬合到生成模型、強化學習等主題，並搭配期末專題進行整合應用。
In this course, I implemented a wide range of statistical and deep learning models using Python and PyTorch, covering topics from function fitting and unsupervised learning to generative models and reinforcement learning, and concluded with a final project.

課程前半介紹了平滑器（Smoothers）、樣條函數（Splines）與廣義加性模型（GAMs），並延伸至無監督式學習（如群集分析）與貝氏統計與階層式模型。
The first part of the course focused on smoothers, splines, and generalized additive models (GAMs), and extended to unsupervised learning such as clustering, as well as Bayesian statistics and hierarchical modeling.

深度學習部分，我學習了卷積神經網路（CNN）、回傳誤差（Backpropagation）、視野與特徵圖可視化（Feature Map Visualization）、注意力熱點（Saliency Maps）與遷移學習等核心概念。
In the deep learning section, I explored convolutional neural networks (CNNs), backpropagation, receptive fields, feature map visualization, saliency maps, and transfer learning techniques.

在處理序列資料的章節中，我實作了 RNN、GRU、LSTM 以進行語言建模任務，並比較其在不同語言嵌入技術（如 Word2Vec）下的表現。
For sequential data tasks, I implemented RNNs, GRUs, and LSTMs for language modeling, and evaluated their performance using various word embedding techniques like Word2Vec.

我也學習了 Transformer、Autoencoder、變分自編碼器（VAE），並進一步深入生成對抗網路（GANs）與深度強化學習（Deep RL）在資料生成與決策中的應用。
I also worked with Transformer models, autoencoders, and variational autoencoders (VAEs), and further explored generative adversarial networks (GANs) and deep reinforcement learning for data generation and decision-making tasks.

微算機原理與實驗

Principle of Microcomputer

National Yang Ming Chiao Tung University
  • Dept. of course offering: Department of Electrical and Computer Engineering
  • Required/Elective: Elective
  • Instructor: 王宜楷

本課程介紹 8051 微控制器架構與應用，涵蓋組合語言、I/O 操作、中斷、計時器、串列通訊等主題，透過理論與實驗課程並重，培養設計與除錯微控制系統的實作能力。
This course introduces the architecture and applications of the 8051 microcontroller, covering assembly language, I/O operations, interrupts, timers, and serial communication. Through combined lectures and hands-on labs, students gain practical skills in designing and debugging microcontroller systems.

本課程每週包含 2 小時講授與 2 小時實驗，內容涵蓋：

  • 8051 單晶片微控器介紹與軟體模擬器、ICE 開發環境實作
  • 指令集與組合語言實作
  • I/O 操作與數位輸入輸出控制
  • Timer/Counter 計時與週期性觸發控制
  • 外部中斷與定時中斷處理
  • 串列傳輸 UART 程式設計與應用
  • 微電腦系統與嵌入式應用概念整合

This lab-based course covers both theoretical concepts and practical implementation of microcontroller systems, enabling students to develop low-level programs in assembly and C, interface with hardware, and analyze timing and interrupt behavior in real time.

📂 課程資料與成果
  • Lab 1 - LED 跑馬燈與霹靂燈效果
    使用 8051 組合語言撰寫 LED 跑馬燈與霹靂燈程式，學習基本 I/O 控制與延遲函數。
    Implemented LED chasing and blinking effects using 8051 assembly to learn basic I/O control and delay routines.
  • Lab 2 - 使用指撥開關控制 LED 與跑馬燈
    透過指撥開關輸入控制 LED 顯示與跑馬燈效果，加強 I/O 控制與 delay 時間調整概念。
    Used DIP switches to control LED outputs and implement a running light pattern, reinforcing I/O and delay control concepts.
  • Lab 3 - 七段顯示器顯示數字與字母
    練習共陽極七段顯示器的控制，顯示 0~9 及 A~F 字元，並使用 TABLE 儲存顯示資料。
    Practiced controlling common-anode 7-segment displays to show digits and letters using lookup TABLE.
  • Lab 4 - 四合一七段顯示器計數器設計
    實作四合一共陽極七段顯示器的倒數計時器，學習多位元輪流顯示與亮度一致的技巧。
    Designed a countdown timer using 4-digit 7-segment display and learned multiplexing techniques for uniform brightness.
  • Lab 5 - 5x7 點矩陣 LED 動畫與掃描控制
    實作點矩陣顯示器控制，設計動畫與跑馬燈效果，深入理解掃描顯示原理與控制流程。
    Controlled 5x7 dot-matrix LED for animated text and marquee effects, understanding scan display logic.
  • Lab 6 - 使用 Timer0 播放樂音旋律
    利用 Timer0 撰寫樂音程式，透過控制 P0.0 輸出不同頻率實現簡單音樂播放。
    Used Timer0 to toggle P0.0 for sound wave generation and implemented simple melody playback.
  • Lab 7 - 矩陣按鍵輸入與按鍵音樂播放
    以矩陣鍵盤控制不同音階播放，結合中斷與Timer製作互動式鍵盤樂器。
    Used matrix keypad and interrupt-driven input to trigger tone generation like a digital musical keyboard.
  • Lab 8 - 串列控制播放樂音與顯示資料
    透過串列通訊接收控制碼播放樂音與顯示字元，強化 8051 串列介面使用能力。
    Used serial communication to receive characters and play corresponding sounds or display content.
  • Lab 9-1 - 使用 TABLE 控制旋轉 LED 顯示中文字
    以資料表控制旋轉 LED 點陣顯示「別當我」中文字，並解決 DPTR 無法存取大於 256 bytes 資料的問題。
    Controlled spinning LED display to show Chinese characters using table lookup, addressing DPTR limitations on large tables.
  • Lab 9-2 - 使用中斷與多段 TABLE 實作跑馬燈
    以中斷驅動方式控制旋轉 LED 模組，透過多段 TABLE 顯示資料實現字元移動動畫。
    Implemented spinning LED marquee using interrupts and multi-segment table for displaying animated Chinese text.
  • Lab 10 – 超音波測距系統設計
    整合 Timer0 與 Counter1 完成超音波距離量測與七段顯示輸出。
    Use Timer0 and Counter1 to measure distance with ultrasonic sensor and display on 7-seg.
  • Lab 11 – 高階中斷控制與優先權應用
    實作多組中斷源與優先順序控制，理解 8051 的中斷處理機制。
    Work with multiple interrupt sources and priority control on the 8051.
  • Lab 12 – 整合控制系統設計實作
    整合蜂鳴器、LCD、鍵盤與中斷功能，設計互動式控制系統。
    Combine buzzer, LCD, keypad, and interrupts to build an interactive control system.
  • Final Project – 音樂播放器系統設計
    以 8051 為核心實作可選曲、顯示與播放功能的嵌入式音樂播放器。
    An embedded music player with song selection, display, and playback features built on 8051.
