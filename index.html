用Python於深度學習

Labs on Python for Deep Learning

National Yang Ming Chiao Tung University
  • Dept. of course offering: Department of Electrical and Computer Engineering
  • Required/Elective: Elective
  • Instructor: 李冕

使用Python完成多種深度學習模型應用，並透過各種實作、分析與比較，了解各種有關深度學習的相關知識。以下為課程的Challenge與我的成果：

Use Python to complete various deep learning model applications, and learn various deep learning related knowledge through various practices, analysis and comparisons. The following are the challenges of the course and my achievements:

  • Challenge 1: Activation Functions for Function Upscaling
    比較不同 activation function（ReLU, GELU 等）在函數近似與外推的效能差異。
    Compare the performance differences of different activation functions (ReLU, GELU, etc.) in function approximation and extrapolation.
  • Challenge 2: Gradient Descent Variants
    探索 Adam、Momentum、SGD 等學習演算法在不同 loss 曲面上的收斂速度。
    Explore the convergence speed of learning algorithms such as Adam, Momentum, SGD on different loss surfaces.
  • Challenge 3: Padding Effects in CNN/RNN/GRU
    比較不同 padding 策略對 NLP 模型的 F1 / AUC 表現。
    Compare the F1 / AUC performance of different padding strategies on NLP models.
  • Challenge 4: Word Embedding with BERT
    以 BERT 模型實作問答與分類任務，理解其 embedding 與 attention 機制。
    Implement question answering and classification tasks using the BERT model to understand its embedding and attention mechanisms.
  • Challenge 5: Dissecting Attention Mechanisms and Training Transformers
    本挑戰深入探索 Transformer 的注意力機制並實作英德翻譯模型。
    This project explores attention in Transformers and builds a translation model.
  • Challenge 6: Embedding Edit Distance with Variational Autoencoders
    使用 VAE 建構潛在空間以反映二進位序列的編輯距離。
    Use VAE to embed binary sequences, making latent distances reflect edit distances.

在這門課中，我透過 Python 與 PyTorch 實作多種統計學與深度學習模型，內容涵蓋從基礎函數擬合到生成模型、強化學習等主題，並搭配期末專題進行整合應用。
In this course, I implemented a wide range of statistical and deep learning models using Python and PyTorch, covering topics from function fitting and unsupervised learning to generative models and reinforcement learning, and concluded with a final project.

課程前半介紹了平滑器（Smoothers）、樣條函數（Splines）與廣義加性模型（GAMs），並延伸至無監督式學習（如群集分析）與貝氏統計與階層式模型。
The first part of the course focused on smoothers, splines, and generalized additive models (GAMs), and extended to unsupervised learning such as clustering, as well as Bayesian statistics and hierarchical modeling.

深度學習部分，我學習了卷積神經網路（CNN）、回傳誤差（Backpropagation）、視野與特徵圖可視化（Feature Map Visualization）、注意力熱點（Saliency Maps）與遷移學習等核心概念。
In the deep learning section, I explored convolutional neural networks (CNNs), backpropagation, receptive fields, feature map visualization, saliency maps, and transfer learning techniques.

在處理序列資料的章節中，我實作了 RNN、GRU、LSTM 以進行語言建模任務，並比較其在不同語言嵌入技術（如 Word2Vec）下的表現。
For sequential data tasks, I implemented RNNs, GRUs, and LSTMs for language modeling, and evaluated their performance using various word embedding techniques like Word2Vec.

我也學習了 Transformer、Autoencoder、變分自編碼器（VAE），並進一步深入生成對抗網路（GANs）與深度強化學習（Deep RL）在資料生成與決策中的應用。
I also worked with Transformer models, autoencoders, and variational autoencoders (VAEs), and further explored generative adversarial networks (GANs) and deep reinforcement learning for data generation and decision-making tasks.
